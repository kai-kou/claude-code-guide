**作成日**: 2026-02-26
**対象**: Claude Code初心者〜中級者
**形式**: Googleスライド
**想定枚数**: 40〜50枚

---

## ストーリーライン

```
導入 → セットアップ → 基本操作 → 設定のカスタマイズ → 自動化 → 効率化TIPS → まとめ
```

ゼロからClaude Codeを使い始めて、自分の環境に合わせたカスタマイズができるようになるまでのストーリー。

## デザイン方針

- シンプルで見やすい配色（白背景 + アクセントカラー）
- コード例はダークテーマのコードブロックで表示
- 実際のスクリーンショットを積極的に使用（assets/）
- 各セクション冒頭にセクション番号とタイトルのディバイダースライド
- 図表・フローチャートで概念を視覚化
